Another Tetris

Controls:

KEY ACTION
LEFT ARROW Move Left
RIGHT ARROW Move Right
Z Rotate Counter-Clockwise
X Rotate Clockwise
DOWN ARROW Soft Drop
C Hard Drop
LEFT SHIFT Hold
Q Quit Game
R Reset Game
P Pause Game
